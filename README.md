# articles

This repository is the OR6107's articles source.

Visit here to browse online : [OR6107.github.io/articles](https://OR6107.github.io/articles)


## Build Setup

### Install mkdocs

```bash
pip install mkdocs
pip install mkdocs-material
```

### Build Site

```bash
mkdocs build
mkdocs serve
```

Visit here to browse offline : [http://127.0.0.1:8000](http://127.0.0.1:8000)
